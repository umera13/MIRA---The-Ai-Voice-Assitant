# 🎙️ MIRA - Machine Intelligence Response Assistant

MIRA (Machine Intelligence Response Assistant) is a Python-based desktop voice assistant that enables users to interact with their computer through voice commands. The assistant uses speech recognition to process user input and performs various system-level tasks such as opening applications, searching the web, controlling system settings, and providing voice responses.

---

## 🚀 Features

- 🎤 Speech-to-text voice command recognition
- 🔊 Text-to-speech responses
- 🌐 Open websites and perform Google searches
- 📖 Search information on Wikipedia
- 💻 Launch desktop applications
- 💬 Send WhatsApp messages
- 📷 Open the system camera
- 🌞 Control screen brightness
- 😂 Tell random jokes
- ⚡ Check internet speed
- 🖥️ User-friendly graphical interface using Tkinter

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/umera13/MIRA-Voice-Assistant.git
```

### 2. Navigate to the Project Directory

```bash
cd MIRA-Voice-Assistant
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Application

```bash
python main.py
```
